# Windows XP Portfolio

This project is a creative and interactive portfolio website designed to mimic the user interface of the Windows XP operating system. It provides a nostalgic and engaging user experience, showcasing the developer's skills and projects in a unique way.

## Features

* **Windows XP Theme:** The whole website looks and feels like a classic Windows XP desktop.
* **Interactive Desktop:** The home page works like a real desktop with icons you can click to open different sections in movable and resizable windows.
* **Start Menu:** A working Start Menu that gives access to all portfolio sections like About Me, Projects, Contact, and more.
* **About Me:** A window that shares information about the developer and their skills.
* **Resume Viewer:** View the developer’s resume directly on the site in PDF format.
* **Projects Showcase:** A section that shows the developer’s projects with links to view live demos.
* **Contact Form:** A form that lets visitors send messages to the developer.
* **Music Player:** Built-in music player that plays background music while browsing.
* **Wallpaper Changer:** Lets users change the desktop background.
* **Command Prompt:** A fun CMD window that allows text-based interaction with the website.
